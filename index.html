maven-in-polyglot-space by daveayan

This is the source code for my experiments with maven in polyglot space - with groovy, scala, jruby, polyglot maven, android etc. This was used in presentations at Columbus Polyglot User Group (03/2011) and QSI Tech Lunch (05/2011)
